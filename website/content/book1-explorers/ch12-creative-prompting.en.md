## ✨ Magic Spell #12 — Creative Prompt Templates

**1. Quiz Creator:**
```
Create 10 multiple choice questions on [Subject].
Level: [appropriate for Grade X]
Each question should have 4 options (A, B, C, D).
Only one option should be correct.
Give the answer key separately at the end.
Write questions in English.
```

---

**2. Story Generator:**
```
Write a short story (250 words).
Setting: A small town in Pakistan, in 2026.
Main character: A 12-year-old curious girl.
Problem: Her science fair project isn't working.
Ending: Positive, with a lesson.
Tone: Fun and exciting.
```

---

**3. Poem Writer:**
```
Write an 8-line poem about [Topic]:
- ABAB rhyme scheme (lines 1 & 3 rhyme, 2 & 4 rhyme)
- [Mood/tone] tone
- Simple words — suitable for a 10-year-old
- In English
```

---

**4. Explainer for Kids:**
```
Explain [concept] to me as if
I'm a curious 10-year-old.
- Keep it to 5 sentences
- Use one funny real-life analogy
- End with a wow fact
```

---

**5. Lesson Plan (For Teachers!):**
```
Create a 45-minute class plan on [Topic].
Class: Grade [X]
Include:
- 5 min: Warm-up activity
- 20 min: Main lesson
- 15 min: Practice exercise
- 5 min: Summary
Use simple materials (just whiteboard/paper).
```

## 🔥 Try It Yourself

Choose any subject for your class and have a quiz built:

```
Create 5 multiple choice questions on [Subject].
Each question should have 4 options (A, B, C, D).
Appropriate level for [class level].
Give the answer key at the end.
Write questions in English.
```

Then give it to a friend to solve!
