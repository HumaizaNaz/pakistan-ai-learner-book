## ✨ Magic Spell #3 — Pehla HTML Document

VS Code mein ek nayi file banao: `meri-website.html`

Ye **exactly** jaisa hai waisa likho:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Meri Pehli Website</title>
  </head>
  <body>
    <h1>Mera Naam Zara Hai!</h1>
    <p>Ye meri pehli website hai.</p>
    <p>Aaj maine HTML seekhi!</p>
  </body>
</html>
```

File save karo (Ctrl+S). Phir Chrome mein drag and drop karo.

**Kya hua? Tumhari website ban gayi!** 🎉

---

**Important Tags:**

| Tag | Kya karta hai | Example |
|-----|--------------|---------|
| `<h1>` | Sab se bada heading | `<h1>Title</h1>` |
| `<h2>` | Chhota heading | `<h2>Sub-title</h2>` |
| `<p>` | Paragraph | `<p>Text yahan</p>` |
| `<html>` | Poori website ka wrapper | Zaroor chahiye |
| `<head>` | Website ki info | Title wagera |
| `<body>` | Jo screen par dikhe | Sab content yahan |

**Zaruri Rule:** Har opening tag `<tag>` ka closing tag `</tag>` hota hai!

## 🔥 Try It Yourself

Apna naam aur apni 3 favourite cheezein likho:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>[Tumhara Naam] ki Website</title>
  </head>
  <body>
    <h1>[Tumhara Naam]</h1>
    <h2>Meri Favourite Cheezein:</h2>
    <p>1. Mujhe [cheez 1] pasand hai.</p>
    <p>2. Mujhe [cheez 2] bhi pasand hai.</p>
    <p>3. Aur [cheez 3] toh meri favourite hai!</p>
  </body>
</html>
```

## 💡 Fun Tip

`<h1>` se `<h6>` tak headings hoti hain:
- `<h1>` — sab se badi (king!)
- `<h2>` — bigg heading
- `<h6>` — sab se choti

Ek page mein sirf ek `<h1>` hona chahiye — jaise ek class mein ek teacher!
