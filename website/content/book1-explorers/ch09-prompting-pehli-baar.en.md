## ✨ Magic Spell #9 — What is Prompt Engineering

**Prompt** = What you write to AI = The instruction you give AI

**Prompt Engineering** = Writing to AI in a way that gets you the exact best answer

---

**Bad prompt vs Good prompt:**

```
❌ Bad prompt:
biryani

✅ Good prompt:
Give me a simple chicken biryani recipe.
Don't list more than 10 ingredients.
Number the steps (1, 2, 3...).
Include the time for each step.
Write in simple English.
```

Try both on Claude — notice how different the answers are!

---

**4 Parts of a Good Prompt (CKFC Formula):**

```
C → CONTEXT     Why do you need this?
K → TASK        What exactly needs to be done?
F → FORMAT      How should the answer look?
C → CONSTRAINTS Any limits or rules?
```

**Example:**

```
[Context]     I'm a 10-year-old student.
[Task]        Tell me about the planets.
[Format]      In 5 bullet points.
[Constraints] Simple words, in English.
```

## 🔥 Try It Yourself

Try these bad prompts on Claude — then try the better version:

**Round 1 — Bad:**
```
write a poem
```

**Round 2 — Better:**
```
Write a 4-line poem about Pakistan
```

**Round 3 — Best:**
```
Write a 4-line poem about Pakistan.
Use simple words, suitable for a 10-year-old.
It should rhyme.
Write it in English.
```

Compare all three answers — what's the difference?
