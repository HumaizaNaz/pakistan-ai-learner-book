## Zara ki Kahani Shuru Hoti Hai...

Karachi ka ek subah. Zara apne school bag mein kuch dhundh rahi thi.

"NOVA," usne apne phone se kaha, "aaj computer class hai. Miss Ayesha chahti hain hum apni website banayein. Mujhe kuch nahi aata."

NOVA ki screen roshan hui. "Koi baat nahi, Zara. Aaj hum shuru karte hain — bilkul zero se."

"Zero se?" Zara ne socha. "Kya zero se bhi kuch ban sakta hai?"

NOVA muskurai. "Sab kuch zero se shuru hota hai."
