## 📖 Story Scene

"Okay NOVA, so what exactly is a website?" Zara asked, opening her browser.

"A website," NOVA said, "is like a folder. It has files inside — text, pictures, code."

"And where does all that live on the internet?"

"On a **server**. A server is basically a powerful computer that stays on 24/7 — so whenever someone opens your website, it can send them the files."

Zara thought about it. "So a browser is like... a delivery person?"

NOVA lit up. "EXACTLY! The browser requests it, the server delivers it, the browser displays it!"

## ✨ Magic Spell #2 — The Secret of a URL

```
https://www.google.com/search?q=pakistan

https   → Secure connection (safe! 🔒)
www     → World Wide Web
google  → Website name (domain)
.com    → Type of website
/search → Address of the page
?q=...  → Search query
```

**What a browser does:**
1. Takes the URL → sends a request to the server
2. Receives HTML, CSS, JS files from the server
3. Combines them and shows you a beautiful webpage

## 🤯 Wow Moment

> That `https://` at the start —
> the `s` stands for **Secure**! Data travels **encrypted** (in a secret code).
> Every time you send something online, encryption protects it! 🔐

## 💡 Fun Tip

**.com .org .pk .edu** — these are domain extensions!
- `.pk` = Pakistani website
- `.edu` = School/University
- `.org` = Non-profit organization
- `.gov` = Government
- `.ai` = AI companies (it was actually Anguilla island's code — AI hijacked it! 😄)
