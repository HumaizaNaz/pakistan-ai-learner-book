## ✨ Magic Spell #5 — CSS ka Pehla Sabak

HTML file mein `<head>` ke andar CSS add karo:

```html
<head>
  <title>Zara ki Rangeen Website</title>
  <style>

    body {
      background-color: #f0f4ff;
      font-family: Arial, sans-serif;
    }

    h1 {
      color: #6c63ff;
      font-size: 36px;
    }

    p {
      color: #2d2d2d;
      font-size: 18px;
    }

  </style>
</head>
```

**CSS ka formula:**
```
selector {
  property: value;
}
```

Matlab: **"kise"** style karna hai → **"kya"** karna hai → **"kitna/kaisa"**

## 🔥 Try It Yourself

Apni website mein ye CSS add karo — phir colors change karte raho jab tak tumhari favourite color scheme na mile:

```html
<style>
  body {
    background-color: #ffeaa7;
    font-family: Georgia, serif;
  }
  h1 {
    color: #6c63ff;
    font-size: 42px;
  }
  p {
    color: #2d2d2d;
    font-size: 20px;
    line-height: 1.6;
  }
</style>
```

**Extra Challenge:** `text-align: center;` add karo `h1` mein — kya hua?

## 🤯 Wow Moment

> CSS colors **16 million se zyada** hain!
> Har color ka ek secret code hota hai — jaise `#ff0000` red hai.
> Ye `#RRGGBB` format hai — Red, Green, Blue ki mixing!
> `#ff0000` = full red, zero green, zero blue = pure red 🔴
