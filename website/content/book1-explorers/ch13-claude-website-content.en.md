## ✨ Magic Spell #13 — AI + HTML = Superpower Workflow

**Step 1: Ask Claude for content**
```
Write the content for an "About Me" section of my personal website.

My info:
- Name: [your name]
- Class: [your class]
- School: [school name]
- Hobbies: [hobby 1], [hobby 2]
- Dream: [your dream]

Format: 2-3 paragraphs, friendly and personal tone,
in English, written as if I'm speaking myself.
```

---

**Step 2: Ask Claude for the HTML too**
```
Create an HTML card for this "About Me" content.
One div.about-card containing an h2 and paragraphs.
Clean code, proper indentation.
Use meaningful CSS class names.
```

---

**Step 3: Ask Claude for CSS**
```
Write CSS for my about-card.
Theme: purple (#6c63ff) and white.
Card with shadow, rounded corners.
Should also be mobile-friendly.
```

---

**Step 4: Debug with Claude**
```
This is my HTML/CSS code — something isn't working:
[paste your code here]

Problem: [what is or isn't happening]
Fix it and explain what was wrong.
```

---

**Complete Workflow Example:**

```html
<!-- Content + code generated by Claude -->
<!DOCTYPE html>
<html>
<head>
  <title>My Website</title>
  <style>
    .about-card {
      max-width: 600px;
      margin: 40px auto;
      background: white;
      border-radius: 16px;
      padding: 32px;
      box-shadow: 0 8px 24px rgba(108, 99, 255, 0.15);
      border-top: 4px solid #6c63ff;
    }
    .about-card h2 {
      color: #6c63ff;
      margin-bottom: 16px;
    }
    .about-card p {
      color: #444;
      line-height: 1.7;
      margin-bottom: 12px;
    }
  </style>
</head>
<body style="background: #f0f4ff; font-family: Arial;">
  <div class="about-card">
    <h2>About Me 👋</h2>
    <p>[Claude-generated content goes here]</p>
    <p>[Second paragraph]</p>
  </div>
</body>
</html>
```
