## ✨ Magic Spell #1 — The Computer City

**How a computer works — simple version:**

```
CPU (Brain)         → thinks, calculates
RAM (Memory)        → remembers things temporarily
Storage (Hard Disk) → saves things permanently
Screen              → shows you everything
Keyboard/Mouse      → how you talk to it
```

**What is the Internet?**

Internet = a huge network of computers. Computers all over the world are connected to each other.

When you open a website:
```
Your computer  →  Sends a request  →  To the server
     ↑                                      ↓
In your browser  ←  The website files come back
```

**How long does it take?**
Less than 0.2 seconds — data travels at the speed of light! ⚡

## 🔥 Try It Yourself

1. Open Chrome
2. Type in the address bar: `google.com` and press Enter
3. Now type: `view-source:https://www.google.com`
4. What do you see? That's the **code** that builds Google's website!
5. Scroll through it — it's all HTML, CSS, and JavaScript!
