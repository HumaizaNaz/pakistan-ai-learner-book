## ✨ Magic Spell #3 — Your First HTML Document

Create a new file in VS Code: `my-website.html`

Write this **exactly** as it appears:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Website</title>
  </head>
  <body>
    <h1>My Name is Zara!</h1>
    <p>This is my first website.</p>
    <p>Today I learned HTML!</p>
  </body>
</html>
```

Save the file (Ctrl+S). Then drag and drop it into Chrome.

**What happened? You just built a website!** 🎉

---

**Important Tags:**

| Tag | What it does | Example |
|-----|-------------|---------|
| `<h1>` | Biggest heading | `<h1>Title</h1>` |
| `<h2>` | Smaller heading | `<h2>Sub-title</h2>` |
| `<p>` | Paragraph | `<p>Text here</p>` |
| `<html>` | Wraps the whole website | Required |
| `<head>` | Website info | Title, etc. |
| `<body>` | What shows on screen | All content goes here |

**Essential Rule:** Every opening tag `<tag>` has a closing tag `</tag>`!

## 💡 Fun Tip

`<h1>` through `<h6>` are your heading options:
- `<h1>` — the biggest (the king!)
- `<h2>` — big heading
- `<h6>` — the smallest

A page should only have one `<h1>` — like one teacher per class!
