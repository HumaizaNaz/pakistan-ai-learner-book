## ✨ Magic Spell #13 — AI + HTML = Superpower Workflow

**Step 1: Content maango Claude se**
```
Meri personal website ke liye "About Me" section ka content likho.

Meri info:
- Naam: [tumhara naam]
- Class: [tumhari class]
- School: [school naam]  
- Hobbies: [hobby 1], [hobby 2]
- Dream: [tumhara sapna]

Format: 2-3 paragraphs, friendly aur personal tone,
English mein, jaise main khud bol rahi hoon.
```

---

**Step 2: HTML bhi Claude se maango**
```
Is "About Me" content ke liye ek HTML card banao.
Ek div.about-card mein h2 aur paragraphs.
Clean code, proper indentation.
CSS class names meaningful rakho.
```

---

**Step 3: CSS Claude se maango**
```
Mere about-card ke liye CSS likho.
Theme: purple (#6c63ff) aur white.
Card mein shadow, rounded corners.
Mobile-friendly bhi hona chahiye.
```

---

**Step 4: Bug fix karo Claude se**
```
Ye mera HTML/CSS code hai — kuch kaam nahi kar raha:
[apna code yahan paste karo]

Problem: [kya ho raha hai ya nahi ho raha]
Fix karo aur explain karo kya galat tha.
```

---

**Complete Workflow Example:**

```html

    }
  </style>

<body style="background: #f0f4ff; font-family: Arial;">
  <div class="about-card">
    <h2>About Me 👋</h2>
    <p>[Claude ka generate kiya hua content yahan]</p>
    <p>[Second paragraph]</p>
  </div>
</body>
</html>
```
