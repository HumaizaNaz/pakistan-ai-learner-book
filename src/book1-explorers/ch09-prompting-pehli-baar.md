## ✨ Magic Spell #9 — Prompt Engineering kya hai

**Prompt** = Jo tum AI ko likhte ho = AI ko diya gaya instruction

**Prompt Engineering** = AI ko iss tarah likhna ke exact best jawab mile

---

**Bad prompt vs Good prompt:**

```
❌ Bad prompt:
biryani

✅ Good prompt:
Mujhe chicken biryani ki simple recipe batao.
Ingredients 10 se zyada mat likho.
Steps numbered karo (1, 2, 3...).
Time bhi batao har step ka.
Roman Urdu mein likho.
```

Dono try karo Claude par — jawab mein kitna fark hai!

---

**Good Prompt ke 4 Parts (CKFC Formula):**

```
C → CONTEXT     Ye cheez kyun chahiye?
K → KAAM        Kya karna hai exactly?
F → FORMAT      Jawab kaise chahiye?
C → CONSTRAINTS Koi limits ya rules?
```

**Example:**

```
[Context]    Main ek 10-saal ki student hoon.
[Kaam]       Mujhe planets ke baare mein batao.
[Format]     5 bullet points mein.
[Constraints] Simple words, Roman Urdu mein.
```

## 🔥 Try It Yourself

Ye bad prompts ko Claude par try karo — phir better version try karo:

**Round 1 — Bad:**
```
poem likho
```

**Round 2 — Better:**
```
Pakistan ke baare mein ek 4-line poem likho
```

**Round 3 — Best:**
```
Pakistan ke baare mein ek 4-line poem likho.
Simple words use karo, 10 saal ke bachche ke liye.
Rhyming honi chahiye.
Roman Urdu mein likho.
```

Teeno jawab compare karo — kya fark hai?
