## ✨ Magic Spell #1 — Computer ki City

**Computer kaise kaam karta hai — simple version:**

```
CPU (Brain)         → sochta hai, calculate karta hai
RAM (Memory)        → temporarily yaad rakhta hai
Storage (Hard Disk) → permanently save karta hai
Screen              → dikhata hai
Keyboard/Mouse      → tum baat karte ho
```

**Internet kya hai?**

Internet = computers ka ek bada network. Duniya bhar ke computers ek doosre se connected hain.

Jab tum koi website open karte ho:
```
Tumhara computer  →  Request bhejta hai  →  Server tak
      ↑                                         ↓
Browser mein  ←  Website ki files wapas aati hain
```

**Kitni der lagti hai?**
0.2 seconds se bhi kam — light ki speed par data travel karta hai! ⚡
