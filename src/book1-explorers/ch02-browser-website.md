## 📖 Story Scene

"Theek hai NOVA, toh website kya hoti hai exactly?" Zara ne poochha, browser open karte hue.

"Ek website," NOVA ne kaha, "ek folder ki tarah hai. Usme files hain — text, pictures, code."

"Aur ye sab internet par kahan rehta hai?"

"**Server** par. Ek server basically ek powerful computer hai jo 24/7 on rehta hai — taake jab bhi koi tumhari website open kare, woh files de sake."

Zara ne socha. "Toh browser ek... delivery boy hai?"

NOVA khush ho gayi. "EXACTLY! Browser request karta hai, server deta hai, browser dikhata hai!"

## ✨ Magic Spell #2 — URL ka Raaz

```
https://www.google.com/search?q=pakistan

https   → Secure connection (safe hai! 🔒)
www     → World Wide Web
google  → Website ka naam (domain)
.com    → Type of website
/search → Page ka address
?q=...  → Search query
```

**Browser ke kaam:**
1. URL lena → server se request karna
2. Server se HTML, CSS, JS files receive karna
3. Inhe mila ke beautiful webpage dikhana

**Popular Browsers:**

| Browser | Company |
|---------|---------|
| Chrome | Google |
| Firefox | Mozilla |
| Safari | Apple |
| Edge | Microsoft |

## 💡 Fun Tip

**.com .org .pk .edu** — ye domain extensions hain!
- `.pk` = Pakistan ki website
- `.edu` = School/University
- `.org` = Non-profit organization
- `.gov` = Government
- `.ai` = AI companies (Antigua island ka code tha — AI ne hijack kar liya! 😄)
