## ✨ Magic Spell #16 — Complete Prompt Engineering Review

**Book 2 mein tumne jo seekha — sab ek jagah:**

---

**1. Zero-Shot — Seedha poochho:**
```
Best for: Simple tasks, factual questions
Example: "Pakistan ki capital kya hai?"
```

**2. Few-Shot — Examples de ke seekhao:**
```
Best for: Specific format ya style chahiye ho
Example: 2-3 examples do, phir naya kaam batao
```

**3. Chain of Thought — Step by step:**
```
Best for: Complex reasoning, math, decisions
Example: "Step by step socho aur explain karo..."
```

**4. Role Prompting — Character do:**
```
Best for: Specific tone, expertise, personality
Example: "Tum ek [expert] ho..."
```

**5. System Prompts — Permanent personality:**
```
Best for: Chatbots, API applications
Example: system field mein poori identity define karo
```

---

**Prompt Engineering Decision Tree:**

```
Kya task simple hai?
├─ Haan → Zero-shot try karo
└─ Nahi → Kya specific format chahiye?
           ├─ Haan → Few-shot use karo
           └─ Nahi → Kya reasoning chahiye?
                      ├─ Haan → Chain of Thought add karo
                      └─ Nahi → Role prompt try karo
```

---

**Advanced Techniques (Preview for Book 3):**

```
• Prompt Chaining    → Ek prompt ka output doosre mein
• Tool Use           → AI ko real tools dena (calculator, search)
• Agent Prompting    → AI khud plan banaye aur execute kare
• RAG                → AI ko apna data do
```

---

**Common Mistakes aur Fixes:**

| Mistake | Fix |
|---------|-----|
| Vague prompt | Specific karo — format, length, tone |
| Ek lambi prompt mein sab kuch | Parts mein todo — conversation karo |
| First jawab accept kar lo | Refine karo — "improve karo", "shorter karo" |
| Prompt save nahi kiya | Working prompts file mein save karo! |
| AI ko blame karo | Prompt ko improve karo |
