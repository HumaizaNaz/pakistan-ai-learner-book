**Kiske liye:** Grade 7–8 | Builders Track
**Duration:** 8 Weeks | 16 Chapters

## Story Aage Chalti Hai...

Ek saal baad. Zara ki website school mein famous ho gayi thi.

Lekin ek din uski classmate Sara ne kaha: "Zara, tumhari website static hai. Mere bhai ki website par ek AI chatbot hai — ussse baat kar sakte hain! Tumhari website sirf dikhti hai."

Zara ne NOVA se poochha: "NOVA, kya main ek real AI chatbot bana sakti hoon?"

NOVA ne kaha: "Haan. Lekin is baar hum ek step aage jaate hain. TypeScript, Next.js, aur... Claude API."

"Claude API?" Zara ne socha.

"Main Claude ka ek secret hissa hoon. API ke zariye tum mujhe **apne code mein** le aa sakti ho. Apni website par. Apna chatbot bana sakti ho."

Zara ki aankhein chamak uthin. "Ab baat karti hoon."
